# 🌴 Palm Oil Leaf Nutrient Stress Detection using YOLO

This project uses Computer Vision and Deep Learning to detect nutrient stress in palm oil leaves.

## 📌 Project Overview
The system identifies nutrient deficiencies from palm oil leaf images and highlights affected areas using YOLO object detection.

## 🚀 Features
- Palm oil leaf disease/stress detection
- YOLO based object detection
- Bounding box visualization
- Confidence score prediction
- Fertilizer recommendation system
- Multilingual support:
  - English
  - Telugu
  - Hindi
  - Tamil

## 🧠 Model
- Algorithm: YOLO
- Framework: Ultralytics
- Deployment: Streamlit Cloud

## 🌿 Detection Classes
- Boron Deficiency
- Nitrogen Deficiency
- Magnesium Deficiency
- Kalium (Potassium) Deficiency
- Healthy Leaf

## 🛠 Technologies Used
- Python
- YOLO
- OpenCV
- Streamlit
- Google Drive Model Hosting

## 📷 Workflow

Image Upload  
↓  
YOLO Detection  
↓  
Nutrient Stress Prediction  
↓  
Recommendation Generation  

## Result
The application helps in early identification of palm oil leaf nutrient problems using AI.
